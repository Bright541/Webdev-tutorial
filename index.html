Understanding HTML

A Simple Guide to HTML Basics

What is HTML?

HTML, or HyperText Markup Language, is the standard language used to create and design web pages. It provides the structure for web content by using a system of tags and attributes.

HTML Tags

HTML uses various tags to define elements on a webpage. Each tag is enclosed in angle brackets (<>). Here are some common tags:

  • <h1> to <h6>: Header tags for headings
  • <p>: Paragraph tag for text
  • <a>: Anchor tag for links
  • <img>: Image tag for displaying images
  • <div>: Division tag for grouping content
  • <span>: Span tag for inline grouping

Sample HTML Structure


<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Page Title</title>
    </head>
    <body>
        <h1>Hello, World!</h1>
        <p>This is a paragraph.</p>
    </body>
</html>

Interesting Facts About HTML

  • HTML was created by Tim Berners-Lee in 1991.
  • HTML stands for HyperText Markup Language.
  • The latest version of HTML is HTML5, which includes new features like video and audio support.
  • HTML is not a programming language but a markup language used for structuring content.
