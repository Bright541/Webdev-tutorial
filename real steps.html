Introduction

Welcome to this tutorial on how to make a website. We'll cover the basics of HTML, CSS, and JavaScript to get you started.

HTML: Structure of Your Website

HTML stands for HyperText Markup Language. It provides the basic structure of a website using elements like headings, paragraphs, links, and images.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Website</title>
</head>
<body>
    <h1>Welcome</h1>
    <p>This is a paragraph.</p>
</body>
</html>

CSS: Styling Your Website

CSS stands for Cascading Style Sheets. It allows you to style HTML elements, such as changing colors, fonts, and layout.

body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
}

h1 {
    color: #333;
}

p {
    color: #666;
}

JavaScript: Adding Interactivity

JavaScript adds interactivity to your website, such as responding to user actions like button clicks.

function greet() {
    alert('Hello, world!');
}

document.getElementById('myButton').addEventListener('click', greet);
