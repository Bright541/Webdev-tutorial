Introduction

Welcome to the HTML Tags Tutorial. Here, you will learn about various HTML tags, their purpose, and how to use them effectively in web development.

HTML Tags

<h1> - Heading 1

The <h1> tag defines the largest heading in HTML. It is typically used for the main title of a page or section.

<h1>This is a heading</h1>

<p> - Paragraph

The <p> tag defines a paragraph. It is used to group sentences and blocks of text.

<p>This is a paragraph.</p>

<a> - Anchor (Link)

The <a> tag defines a hyperlink, which is used to link to other pages or resources.

<a href="https://example.com">This is a link</a>

<img> - Image

The <img> tag is used to embed images in a webpage.

<img src="image.jpg" alt="Description">

<ul> and <li> - Unordered List

The <ul> tag defines an unordered (bulleted) list. The <li> tag defines a list item.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<table>, <tr>, <th>, <td> - Table

The <table> tag defines a table. The <tr> tag defines a table row, <th> defines a header cell, and <td> defines a standard cell.

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>
